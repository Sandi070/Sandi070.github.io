Proses Pembentukan Senyawa Ion

Pembentukan NaCl (Natrium Klorida)

Natrium (Na) melepas satu elektron dari kulit terluarnya untuk membentuk ion Na+. Klor (Cl) menerima satu elektron dan membentuk ion Cl-. Tarikan elektrostatik antara Na+ dan Cl- membentuk senyawa ion NaCl.

Na → Na+ + e-
Cl + e- → Cl-
Na+ + Cl- → NaCl

Pembentukan CaCl₂ (Kalsium Klorida)

Kalsium (Ca) melepas dua elektron untuk membentuk ion Ca2+. Dua atom klor masing-masing menerima satu elektron dan membentuk dua ion Cl-. Tarikan antara ion Ca2+ dan dua ion Cl- membentuk senyawa CaCl₂.

Ca → Ca2+ + 2e-
2Cl + 2e- → 2Cl-
Ca2+ + 2Cl- → CaCl₂

Pembentukan Na₂O (Natrium Oksida)

Dua atom natrium (Na) masing-masing melepas satu elektron untuk membentuk dua ion Na+. Oksigen (O) menerima dua elektron untuk membentuk ion O2-. Tarikan antara dua ion Na+ dan ion O2- membentuk senyawa Na₂O.

2Na → 2Na+ + 2e-
O + 2e- → O2-
2Na+ + O2- → Na₂O

Pembentukan CaO (Kalsium Oksida)

Kalsium (Ca) melepas dua elektron untuk membentuk ion Ca2+, dan oksigen (O) menerima dua elektron untuk membentuk ion O2-. Tarikan elektrostatik antara Ca2+ dan O2- membentuk senyawa CaO.

Ca → Ca2+ + 2e-
O + 2e- → O2-
Ca2+ + O2- → CaO

Pembentukan AlCl₃ (Aluminium Klorida)

Aluminium (Al) melepas tiga elektron untuk membentuk ion Al3+. Tiga atom klor (Cl) masing-masing menerima satu elektron untuk membentuk tiga ion Cl-. Ikatan antara Al3+ dan Cl- membentuk senyawa AlCl₃.

Al → Al3+ + 3e-
3Cl + 3e- → 3Cl-
Al3+ + 3Cl- → AlCl₃

Pembentukan Na₃P (Natrium Fosfida)

Tiga atom natrium (Na) masing-masing melepas satu elektron untuk membentuk tiga ion Na+. Fosfor (P) menerima tiga elektron untuk membentuk ion P3-. Ikatan antara Na+ dan P3- membentuk senyawa Na₃P.

3Na → 3Na+ + 3e-
P + 3e- → P3-
3Na+ + P3- → Na₃P

Pembentukan Al₂O₃ (Aluminium Oksida)

Dua atom aluminium (Al) masing-masing melepas tiga elektron untuk membentuk dua ion Al3+. Tiga atom oksigen (O) masing-masing menerima dua elektron untuk membentuk tiga ion O2-. Tarikan elektrostatik antara Al3+ dan O2- membentuk senyawa Al₂O₃.

2Al → 2Al3+ + 6e-
3O + 6e- → 3O2-
2Al3+ + 3O2- → Al₂O₃
