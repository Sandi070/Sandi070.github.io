Penyetaraan Reaksi Kimia (Soal a–n)

a) __K + __H2O → __H2 + __KOH
Jawaban: 2K + 2H2O → H2 + 2KOH
Pembahasan: 2 mol K menghasilkan 2 mol KOH dan 1 mol H2. Jumlah K dan H disetarakan.
b) __CuO + __NH3 → __H2O + __N2 + __Cu
Jawaban: 3CuO + 2NH3 → 3Cu + N2 + 3H2O
Pembahasan: 3 CuO direduksi menjadi Cu, NH3 teroksidasi jadi N2. Jumlah Cu, N, dan H disetarakan.
c) __Al + __HCl → __AlCl3 + __H2
Jawaban: 2Al + 6HCl → 2AlCl3 + 3H2
Pembahasan: 1 mol Al bereaksi dengan 3 mol HCl untuk membentuk AlCl3 dan H2.
d) __ZnS + __O2 → __ZnO + __SO2
Jawaban: 2ZnS + 3O2 → 2ZnO + 2SO2
Pembahasan: Disetarakan berdasarkan jumlah atom Zn, S, dan O di kiri dan kanan.
e) __NH4Cl + __Ca(OH)2 → __CaCl2 + __NH3 + __H2O
Jawaban: 2NH4Cl + Ca(OH)2 → CaCl2 + 2NH3 + 2H2O
Pembahasan: 2 mol NH4Cl menghasilkan 2 mol NH3, air, dan garam dari Ca.
f) __C4H10 + __O2 → __CO2 + __H2O
Jawaban: 2C4H10 + 13O2 → 8CO2 + 10H2O
Pembahasan: Jumlah atom C = 8, H = 20, O total = 26. Maka 13 O2.
g) __H3PO4 + __NaOH → __Na3PO4 + __H2O
Jawaban: H3PO4 + 3NaOH → Na3PO4 + 3H2O
Pembahasan: Netralisasi 3H dari asam dengan 3OH dari basa.
h) __C2H2 + __O2 → __CO2 + __H2O
Jawaban: 2C2H2 + 5O2 → 4CO2 + 2H2O
Pembahasan: C = 4, H = 4, O total di kanan = 10 → 5 O2.
i) __KClO3 → __KCl + __O2
Jawaban: 2KClO3 → 2KCl + 3O2
Pembahasan: Jumlah O di kiri = 6 → di kanan 3O2.
j) __C2H6 + __O2 → __CO2 + __H2O
Jawaban: 2C2H6 + 7O2 → 4CO2 + 6H2O
Pembahasan: C = 4, H = 12 → 6H2O. O total = 10.
k) __N2 + __H2 → __NH3
Jawaban: N2 + 3H2 → 2NH3
Pembahasan: 2N dan 6H di kanan → 1N2 dan 3H2.
l) __N2H4 + __O2 → __N2 + __H2O
Jawaban: N2H4 + O2 → N2 + 2H2O
Pembahasan: N dan H disetarakan, O dari 2H2O = 2 → 1 O2.
m) __Na + __Cl2 → __NaCl
Jawaban: 2Na + Cl2 → 2NaCl
Pembahasan: Cl2 = 2 Cl atom, dibutuhkan 2 Na → 2 NaCl.
n) __Fe + __O2 → __Fe2O3
Jawaban: 4Fe + 3O2 → 2Fe2O3
Pembahasan: 4 Fe dan 6 O → dari 3O2.
